## рџ“ђ Mathematical Background :
This implementation is based on the mathematical formulation presented in:

> **Mathematical Foundations for Scalable Tic-Tac-Toe: Vector Parametric Edition (II)**

Instead of explicitly checking every possible winning combination, the program mathematically constructs all winning rows, columns, and diagonals using **Vector Parametric Equations**. This approach naturally scales to any **N Г— N** board while keeping the implementation concise, reusable, and mathematically elegant.

The software serves as a practical implementation of the concepts developed in the accompanying mathematical work.
